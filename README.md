## 🚀 О проекте

**Aqbobek Lyceum Platform** — это многоролевой портал (Ученик, Учитель, Родитель, Админ), решающий проблему разрозненности школьных данных. Система интегрирует успеваемость из BilimClass, достижения учащихся и сложные административные задачи в один интуитивно понятный интерфейс.

### 🔥 Killer Features (Наши козыри)

* **🧠 AI-Тьютор и Предиктивная аналитика:** Система анализирует оценки и с вероятностью до 80% предсказывает риски по СОЧ/СОР, предлагая конкретные видеолекции (YouTube) для закрытия пробелов.
* **🗓️ Smart Schedule (Hardcore Module):** Алгоритм автоматической генерации расписания без конфликтов учителей и кабинетов. Поддержка «лент», пар и динамического перестроения при болезни учителя.
* **🛡️ LLM Privacy Shield:** Уникальная система безопасности. Все персональные данные (ФИО) хешируются по алгоритму SHA-256 перед отправкой в нейросеть. Конфиденциальность данных учащихся на уровне Zero-Trust.
* **📈 Реактивный MMR и Геймификация:** Ученики получают/теряют MMR в реальном времени на основе оценок. Внедрена система «Карты навыков» (Skill Map) для визуализации прогресса.
* **📺 Kiosk Mode:** Специальный режим «Интерактивной стенгазеты» для коридорных панелей с автоскроллом, живым расписанием и топ-листом учеников.
* **🌍 Тотальная локализация:** Полная поддержка RU/KK/EN (i18n) с корректной школьной терминологией.

## 🛠 Технологический стек

* **Frontend:** Next.js 15 (App Router), React 19, Tailwind CSS v4.
* **AI:** Google Gemini Flash 3 (через API), кастомные алгоритмы предиктивного анализа.
* **State Management:** React Context + LocalStorage Sync (для мгновенного обновления MMR).
* **Security:** SHA-256 Hashing, Environment Variable Protection.
* **Icons:** Lucide-React.

## ⚙️ Установка и запуск

1.  **Клонируйте репозиторий:**
    ```bash
    git clone [https://github.com/your-repo/second-team-hack.git](https://github.com/your-repo/second-team-hack.git)
    ```
2.  **Установите зависимости:**
    ```bash
    npm install
    ```
3.  **Настройте переменные окружения:**
    Создайте файл `.env.local` и добавьте ваш ключ:
    ```env
    NEXT_PUBLIC_GEMINI_API_KEY=your_api_key_here
    ```
4.  **Запустите сервер разработки:**
    ```bash
    npm run dev
    ```
